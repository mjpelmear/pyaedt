SBR+: HFSS to SBR+ coupling
---------------------------
This example shows how you can use PyAEDT to create an HFSS SBR+ project from an
HFSS antenna and run a simulation.

~~~~~~~~~~~~~~~~~~~~~~~~

directory.

Create setup and solve
